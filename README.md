# Wittysoft Barbershop Management System

Una solución integral de grado empresarial diseñada para la automatización y gestión optimizada de barberías modernas. Este sistema no solo gestiona citas, sino que centraliza la operación financiera, el control de inventario y la experiencia del cliente en una plataforma única, rápida y escalable.

# Características Principales
* **Motor de Reservas Inteligente:** Gestión de turnos en tiempo real con prevención de solapamientos.
* **Gestión de Inventario (SaaS Ready):** Control de stock de productos y suministros de barbería.
* **Dashboard Financiero:** Reportes de ingresos diarios, comisiones por barbero y proyecciones.
* **Arquitectura PWA:** Experiencia nativa en móviles con capacidades offline y notificaciones push.
* **Multi-tenant (Opcional):** Preparado para escalar de una sola sucursal a una franquicia.
