# Stack Tecnológico

### Frontend (User Experience)
* **Next.js 15 (App Router):** Para renderizado híbrido y optimización de rutas.
* **React Query (TanStack):** Gestión de estado asíncrono y caché de datos.
* **Tailwind CSS + Shadcn/UI:** Componentes atómicos de alta calidad y diseño minimalista.
* **Lucide React:** Iconografía ligera.

### Backend & Data (Architecture)
* **NestJS:** Framework de Node.js robusto para una arquitectura modular y escalable.
* **PostgreSQL:** Base de datos relacional para garantizar integridad en transacciones financieras.
* **Prisma ORM:** Typesafe database access para evitar errores en tiempo de ejecución.
* **TypeScript:** Tipado estricto en todo el monorepo.

### DevOps & Extras
* **Docker:** Containerización para despliegue consistente.
* **Nginx:** Proxy inverso para alta disponibilidad.
* **n8n / Webhooks:** Automatización de recordatorios vía WhatsApp/Email.
* **PWA (next-pwa):** Para instalación en dispositivos móviles sin pasar por App/Play Store.
